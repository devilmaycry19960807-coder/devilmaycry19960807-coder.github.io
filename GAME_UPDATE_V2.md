# 游戏重大更新 V2

## ✅ 已完成的所有修改

### 1. 🎯 降低前50层难度
- 怪物生命、攻击、防御全部大幅降低
- 正常情况下90%几率可以平稳度过前50层
- 适合新手和休闲玩家

### 2. 🏪 自动商店系统
- 每过20-40层自动触发商店（70%概率）
- 商店展示6件装备
- 总价约为玩家当前金币的90%-120%
- 可根据需要购买，不强制购买
- 离开商店后记录位置，继续探索

### 3. 💀 简化死亡逻辑
- 死亡后自动回到上一层
- 状态完全恢复（满血满法力）
- 不弹出任何对话框
- 继续自动战斗

### 4. 📊 排行榜改进
- 新增"📤 上传当前成绩"按钮
- 手动上传到排行榜
- 显示当前层数和金币
- 排行榜存储在localStorage

### 5. 💾 自动存档读取
- 打开游戏时自动读取上次存档
- 只要不删除存档，永远不会丢失进度
- 退出游戏自动保存

## 🎮 游戏玩法说明

1. **开始游戏** - 点击"开始自动战斗"
2. **自动探索** - 自动战斗、自动升级、自动装备
3. **自动商店** - 每20-40层有70%概率进入商店
4. **死亡回退** - 死后回到上一层继续
5. **排行榜** - 随时手动上传成绩

## 🔧 技术细节

- **存档机制**: localStorage存储，自动保存/读取
- **商店生成**: 随机6件装备，总价动态计算
- **难度曲线**: 前50层难度显著降低，后期保持原样
- **数据持久化**: 排行榜和存档本地存储

## 📝 注意事项

1. 存档保存在浏览器localStorage中
2. 清除浏览器缓存会删除存档
3. 排行榜也是本地存储，不同设备不共享
4. 要保护存档，不要清除浏览器数据
