# ⚔️ 无尽地下城 - Roguelike冒险

欢迎来到无尽地下城！这是一款文字Roguelike冒险游戏，拥有无限的成长空间。

## 🎮 游戏特点

- 🏰 **无限层数地下城**，难度逐渐提升
- 🎯 **多种怪物**：普通、精英、BOSS
- 💪 **三大基础属性**：力量、敏捷、智力系统
- 🎒 **装备系统**：8个装备槽位，4种稀有度
- ✨ **法术系统**：10种法术，升级自动学习
- ★ **遗物系统**：可无限叠加的被动效果
- 💾 **存档系统**：随时保存和读取进度
- ⚡ **自动战斗**：一键自动探索和战斗

## 🎯 游戏机制

### 属性系统
- **力量**: 增加攻击力和生命值
- **敏捷**: 增加防御、闪避、暴击率
- **智力**: 增加法力值、法力恢复、法术伤害

### 装备系统
- **装备槽位**: 头盔、铠甲、左手、右手、护手、靴子、首饰1、首饰2
- **稀有度**: 普通（灰色）、优秀（绿色）、稀有（蓝色）、传说（金色）

### 法术系统
- **攻击法术**: 火球术、冰刺、闪电链、火焰爆裂、陨石坠落
- **治疗法术**: 治疗术、圣光治疗
- **增益法术**: 魔法护盾、闪现、力量增幅
- 升级后自动学习新法术

## 🕹️ 操作说明

- ▶️ **开始自动战斗**: 自动探索和战斗
- ⚔️ **手动攻击**: 手动攻击当前怪物
- 🏪 **随机装备**: 花费金币随机获得装备或遗物
- 💚 **治疗**: 花费50金币恢复30%HP
- 💾 **存档**: 保存当前游戏进度
- 📂 **读档**: 读取之前的存档
- 🏃 **逃跑**: 遇到强敌时可尝试逃跑

## 👹 怪物类型

### 普通怪
- 深渊魔鼠、暗影蜘蛛、腐烂僵尸、骨矛守卫、虚空行者、熔岩史莱姆

### 精英怪
- 深渊领主之影、死亡骑士、暗炎魔像、虚空吞噬者

### BOSS（每10层）
- 深渊魔王·索拉瑞斯
- 永恒守卫·阿瑞斯
- 虚空之主·泽拉图
- 炎龙之王·伊格尼斯
- 死亡女神·海拉

## 💡 游戏玩法

1. 点击"开始自动战斗"或手动探索
2. 遭遇怪物后进行回合制战斗
3. 击败怪物获得金币、经验、装备
4. 使用金币购买装备或治疗
5. 挑战更深层的地下城

## 🚀 开始游戏

直接在浏览器中打开 `index.html` 即可开始游戏！
