## 💰 重新设计的变现方案

### 核心思路：广告激励 + 数值付费

#### 1. 激励视频广告（主要收入来源）

**广告点设计：**

| 功能 | 当前状态 | 广告版 | 说明 |
|------|---------|--------|------|
| 复活 | 死后回退一层 | 看广告原地复活 | 保留进度，不损失层数 |
| 宝箱奖励 | 固定金币 | 看广告翻倍金币 | 增加额外奖励 |
| 商店刷新 | 免费 | 看广告刷新出更好的装备 | 提升出稀有装备概率 |
| 自动战斗速度 | 1.2秒/回合 | 看广告2x速度 | 提升刷怪效率 |
| 每日签到 | 无 | 看广告补签 | 错过奖励的补救 |
| 抽卡强化 | 无 | 看广告额外抽取 | 多获得一件装备/遗物 |

**预估收益：**
- 日活用户：10,000人
- 人均日观看：5-8次广告
- eCPM：¥20-40（国内激励视频）
- **月收入：30万-96万**

#### 2. Banner广告（补充收入）

**广告位：**
- 页面底部固定Banner
- 游戏结束弹窗顶部Banner
- 商店面板侧边Banner
- 排行榜页面Banner

**预估收益：**
- 日活用户：10,000人
- 人均展示：50-100次/天
- eCPM：¥5-10
- **月收入：1.5万-30万**

#### 3. 插屏广告（关键节点）

**触发时机：**
- 击败BOSS后（可选择关闭或看广告）
- 升级后（展示新能力，可看广告获得额外属性）
- 每10层休息点（看广告全血+全法力恢复）

**预估收益：**
- 日活用户：10,000人
- 人均展示：3-5次/天
- eCPM：¥10-20
- **月收入：9万-60万**

## 🎮 游戏机制改造

### 1. 新增"每日任务系统"

```javascript
每日任务示例：
- [ ] 探索5次 → 奖励：100金币
- [ ] 击败10只怪物 → 奖励：随机装备（看广告翻倍）
- [ ] 使用3次法术 → 奖励：50法力恢复药水（看广告获得）
- [ ] 购买1次装备 → 奖励：商店刷新券（看广告获得）
```

**变现点：**
- 任务奖励可看广告翻倍
- 特殊任务需要看广告解锁

### 2. 新增"限时活动"

**每日BOSS挑战：**
- 每天刷新特殊BOSS
- 击败后获得特殊遗物
- 挑战失败可看广告复活继续挑战

**周末双倍活动：**
- 周末双倍经验/金币
- 看广告额外延长2小时

### 3. 新增"排行榜系统"

**多维度排行：**
- 深度排行榜（最高层数）
- 速度排行榜（最快到达100层）
- 装备评分排行榜
- 连胜排行榜

**变现点：**
- 查看他人装备详情需看广告
- 挑战第一名看广告获得额外加成

### 4. 新增"仓库系统"

**当前痛点：**
- 无法保留旧装备
- 好装备换下来就没了

**解决方案：**
- 添加仓库（5-10格初始，看广告扩容）
- 可查看、对比装备
- 看广告增加仓库容量

```javascript
            addLog('复活了！继续战斗！', 'info');

            // 正常死亡流程

    adProvider.showRewardedAd({

    });
}
```
