## 📝 最简化填写步骤

### Step 1: 填写 4 个必填框

**位置：页面顶部**

```
Game title: Endless Dungeon - 无尽地下城

Short description: A text-based Roguelike adventure with endless floors and equipment grinding. Free to play!

Kind of project: 选择 HTML

Game URL: https://devilmaycry19960807-coder.github.io/
```

---

### Step 2: 选择发布状态

**位置：页面中间**

```
Release status: 选择 Released

Visibility: 选择 Public
```

---

### Step 3: 点击发布

**位置：页面底部**

找到按钮，文字可能是：
- "Publish page"
- "Create page"
- "Save changes"

点击即可！

## 💡 小贴士

1. **滚动到最底部**：发布按钮通常在页面最底部
2. **先预览**：有些版本有 "View page" 按钮，可以先预览
3. **链接正确**：确保 Game URL 能访问
4. **保存草稿**：如果不确定，可以先保存为 Draft
