# 超级简单的 itch.io 提交方案

**最省事的方法 - 只需要打字，不需要设计！**

---

## 🎯 方案：不需要封面图

itch.io 可以**不传封面图**，使用默认样式！

---

## 📝 最简单的步骤

### 第1步：打开网站（1分钟）

```
https://itch.io/
```

点击右上角 **"Register"**

---

### 第2步：注册（2分钟）

填写：
- Username（用户名）：随便填，比如 `endlessdungeon`
- Email（邮箱）：填你的邮箱
- Password（密码）：随便设一个
- 点击 **"Register"**

---

### 第3步：创建游戏（1分钟）

登录后：
1. 点击右上角头像
2. 点击 **"Upload game"**
3. 点击 **"Create new game page"**

---

### 第4步：填写信息（5分钟）

#### 必填项：

**Game title（游戏标题）：**
```
Endless Dungeon
```

**Short description（简短描述）：**
```
A text-based Roguelike adventure with endless floors and auto-battle. Free!
```

**Kind of project（项目类型）：**
选择 **"HTML"**

**Game URL（游戏链接）：**
```
https://devilmaycry19960807-coder.github.io/
```

#### 选填项（可以留空）：

- Long description（详细描述）：直接留空
- Tags（标签）：直接留空
- Screenshots（截图）：直接不传
- Cover image（封面图）：直接不传

---

### 第5步：发布（1分钟）

1. 拉到页面最底部
2. 点击 **"View page"** 预览
3. 如果能玩，点击 **"Publish"**

---

## ✅ 完成！

**总共耗时：10分钟**

就这么简单！
