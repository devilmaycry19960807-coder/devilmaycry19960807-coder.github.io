# 游戏攻略合集

## 📘 新手指南

### 第一阶段：开局（1-10层）

**开局特点：**
- 你的属性是超强状态，可以轻松通过前10层
- 所有怪物都打不过你
- 这是一个熟悉游戏机制的阶段

**操作建议：**
1. ✅ 开启"自动战斗"
2. ✅ 遇到宝箱全部打开
3. ✅ 不要逃跑，直接打
4. ✅ 金币先攒着，不要急着花

**10层BOSS：**
- 一定会掉落传说武器
- 拿到传说武器是开局的关键
- 建议截图留念！

**属性分配：**
- 开局属性已经很强，不需要调整
- 等到10层以后再考虑属性加点

**装备选择：**
- 前10层掉落的装备都是普通的
- 不用太在意，等10层BOSS的传说武器

---

### 第二阶段：发展期（11-30层）

**难度提升：**
- 怪物开始变强
- 需要合理分配属性
- 装备开始变得重要

**属性分配建议：**
```
新手推荐：力量流
- 力量：60%
- 敏捷：20%
- 智力：20%

优势：简单粗暴，容错率高
劣势：后期疲软
```

**装备选择优先级：**
1. **武器** - 攻击力最重要
2. **铠甲** - 增加血量和防御
3. **头盔** - 增加属性
4. **首饰** - 增加攻击或血量

**商店策略：**
- 金币优先买装备
- 每层遇到商店都去看看
- 批量购买更划算（x5, x10）

**战斗策略：**
- 还是很简单，自动战斗就行
- 遇到太难打的怪可以考虑逃跑
- 血量低的时候自动会用治疗法术

---

### 第三阶段：挑战期（31-50层）

**难度明显提升：**
- 精英怪物变多
- 需要更好的装备
- 开始考验装备搭配

**属性分配调整：**
```
力量流继续：
- 力量：50%
- 敏捷：30% (增加闪避)
- 智力：20%
```

**装备搭配原则：**
1. **优先装备高稀有度**：蓝色>绿色>灰色
2. **属性优先级**：攻击力>血量>其他
3. **套装考虑**：同类型的装备可以叠加属性

**商店策略：**
- 金币多的时候可以x20批量购买
- 多买装备，不要省
- 治疗也可以考虑

**战斗技巧：**
- 看情况选择逃跑
- 利用法术自动触发
- 注意血量，及时治疗

**20层、30层BOSS：**
- 会掉落传说装备
- 尽量拿到
- 如果打不过可以逃跑，等层数高了再回来

---

### 第四阶段：高级期（51-100层）

**难度很高：**
- 普通怪物也很强
- 精英怪物很多
- 需要精良的装备

**属性分配：**
```
中期力量流：
- 力量：40%
- 敏捷：40% (闪避很重要)
- 智力：20%
```

**装备策略：**
1. **追求蓝色装备**：这个阶段蓝色是主力
2. **传说装备看运气**：能刷到最好
3. **装备搭配**：攻击+血量平衡

**商店策略：**
- 必须批量购买（x50或x100）
- 装备和治疗都要买
- 不能省金币

**战斗策略：**
- 逃跑是正常策略，不要硬刚
- 法术很关键，高等级法术伤害高
- 注意生命值，及时治疗

**40层、50层BOSS：**
- 很难打，建议堆属性后再来
- 掉落传说装备
- 可以看广告复活打

---

### 第五阶段：专家期（100层+）

**地狱难度：**
- 所有怪物都很强
- 需要完美的装备
- 考验操作和策略

**属性分配：**
```
后期均衡流：
- 力量：35%
- 敏捷：35%
- 智力：30%

闪避和法术都很重要！
```

**装备要求：**
- 全身蓝色装备起步
- 传说装备越多越好
- 装备属性要平衡（攻击+血量+防御+闪避）

**商店策略：**
- 每次都x100批量购买
- 大量买装备和药水
- 不用省金币

**战斗策略：**
- 灵活运用逃跑
- 法术是主力输出
- 注意生命值管理

**死亡处理：**
- 装备不错可以看广告复活
- 装备一般就重开
- 每次重开都是新体验

---

## 🎯 三大流派详解

### 1. 力量流（适合新手）

**核心属性：** 力量

**属性分配：**
```
前期：力量80%，敏捷10%，智力10%
中期：力量60%，敏捷25%，智力15%
后期：力量45%，敏捷35%，智力20%
```

**装备选择：**
- 武器：高攻击力
- 铠甲：高血量
- 头盔：高力量
- 其他：优先力量、血量、攻击

**优点：**
- ✅ 简单粗暴，容易理解
- ✅ 前期优势明显
- ✅ 容错率高
- ✅ 适合休闲玩家

**缺点：**
- ❌ 后期疲软
- ❌ 缺乏爆发
- ❌ 不灵活

**适合人群：**
- 新手玩家
- 不喜欢思考的玩家
- 休闲玩家

---

### 2. 敏捷流（适合进阶）

**核心属性：** 敏捷

**属性分配：**
```
前期：敏捷70%，力量20%，智力10%
中期：敏捷60%，力量25%，智力15%
后期：敏捷50%，力量30%，智力20%
```

**装备选择：**
- 武器：高暴击伤害
- 铠甲：高闪避
- 头盔：高敏捷
- 其他：全堆闪避、暴击、暴击伤害

**优点：**
- ✅ 爆发高
- ✅ 闪避爽
- ✅ 高风险高收益
- ✅ 看脸就有无限可能

**缺点：**
- ❌ 看脸，不稳定
- ❌ 脆皮
- ❌ 操作要求高

**适合人群：**
- 欧皇
- 喜欢高风险高收益的玩家
- 进阶玩家

---

### 3. 智力流（适合高手）

**核心属性：** 智力

**属性分配：**
```
前期：智力70%，力量20%，敏捷10%
中期：智力60%，力量25%，敏捷15%
后期：智力50%，力量30%，敏捷20%
```

**装备选择：**
- 武器：高法术伤害
- 铠甲：高法力值
- 头盔：高智力
- 其他：全堆智力、法术伤害

**法术搭配：**
- 优先学习攻击法术：火球术、冰刺、闪电链
- 治疗法术保命：治疗术
- 增益法术辅助：魔法护盾、力量增幅

**优点：**
- ✅ 远程输出
- ✅ AOE伤害高
- ✅ 法术多样
- ✅ 策略性强

**缺点：**
- ❌ 蓝耗大
- ❌ 前期弱
- ❌ 操作复杂

**适合人群：**
- 喜欢策略的玩家
- 高手玩家
- 喜欢法术的玩家

## 💰 金币管理策略

### 金币获取：
1. 击败怪物
2. 开启宝箱
3. 完成层数（每层奖励）

### 金币使用优先级：

**高优先级：**
1. ✅ 批量购买装备（x100）
2. ✅ 治疗药水（血量低时）
3. ✅ 遗物（如果运气好买到）

**中优先级：**
1. 🔄 单次购买装备（金币少时）
2. 🔄 少量治疗药水

**低优先级：**
1. ⏸️ 不要频繁单次购买
2. ⏸️ 不要浪费金币

### 批量购买建议：

```
金币 < 500：单次购买
500-2000：x5批量购买
2000-10000：x10批量购买
10000-50000：x20批量购买
50000+：x50或x100批量购买
```

## ⚔️ 战斗策略

### 自动战斗模式：
- 适合大多数情况
- 省心省力
- 自动触发法术

### 手动战斗模式：
- 精英怪物可以手动
- BOSS战建议手动
- 更好的法术时机把握

### 逃跑策略：
- 遇到打不过的怪就跑
- 不要硬刚，保命重要
- 跑了可以换装备再来

### 法术使用：
- **攻击法术**：每次攻击自动释放最高级的
- **治疗法术**：血量<50%时自动释放
- **增益法术**：根据情况使用

## 💎 装备刷取技巧

### 什么时候刷装备：
- 装备稀有度低的时候
- 属性不好、总是失败的时候
- 想追求完美装备的时候

### 刷装备方法：
1. 找到层数合适的怪物
2. 反复购买装备
3. 遇到好的就换上
4. 不好的继续刷

### 什么时候该换装备：
- 看到更高稀有度的装备
- 属性更好的同稀有度装备
- 更适合你流派的装备

### 装备搭配原则：
1. 同类型装备可以叠加属性
2. 攻击+血量+防御+闪避平衡
3. 不要只堆一个属性

---

## 🏆 排行榜攻略

### 上榜条件：
1. 层数越高越好
2. 同层数比金币
3. 金币越多排名越高

### 攻略要点：
1. 活得越久越好
2. 多杀怪多拿金币
3. 不要频繁死亡
4. 装备越好能活越久

### 冲榜策略：
- 不要频繁重开
- 每次都尽力冲更高层数
- 多用复活功能

## ❓ 常见问题

### Q：多久能通关100层？
A：新手可能需要一周，熟练后2-3天可以。

### Q：传说装备掉率多少？
A：很低，看运气。刷100件装备可能出1件。

### Q：死亡后损失什么？
A：接受死亡则重开；看广告复活则保留装备获得Buff。

### Q：层数有上限吗？
A：没有，理论上可以无限玩。

### Q：哪个流派最简单？
A：力量流最简单，推荐新手。

### Q：怎么快速升级？
A：多杀怪，多打层数，经验是自动的。
