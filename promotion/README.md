# 游戏推广素材包

本文件夹包含所有免费推广所需的素材和模板。

- `game_long_description.md` - 详细版游戏介绍（用于官网、应用商店）
- `game_highlights.md` - 游戏亮点提炼（用于广告文案）

### 2. 社交媒体素材
- `weibo_post_template.md` - 微博发帖模板
- `zhihu_answer_template.md` - 知乎回答模板
- `tieba_post_template.md` - 贴吧发帖模板
- `bilibili_video_script.md` - B站视频脚本
- `douyin_video_script.md` - 抖音短视频脚本

### 3. 游戏攻略内容
- `beginner_guide.md` - 新手攻略
- `strategy_guide.md` - 进阶攻略

### 4. 营销推广材料

## 🚀 推广计划建议

### 第一阶段：基础推广（1-2周）
1. 在相关游戏贴吧、论坛发帖
2. 在知乎、微博分享游戏介绍
3. 录制简单游戏视频上传B站

### 第二阶段：内容营销（3-4周）
1. 发布游戏攻略系列文章
2. 邀请小游戏主播试玩
3. 在游戏社区持续更新游戏进展

### 第三阶段：扩大影响（持续）
1. 根据玩家反馈优化游戏
2. 举办游戏活动
3. 建立玩家社群
