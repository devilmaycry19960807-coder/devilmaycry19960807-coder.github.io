# 游戏封面图制作指南

itch.io 需要一张封面图（Cover Image），我来告诉你如何制作。

---

## 📐 封面图规格要求

### 推荐尺寸：
```
640 x 360 像素（16:9 比例）
```

### 备选尺寸：
```
1280 x 720 像素
1920 x 1080 像素
```

### 支持格式：
```
PNG（推荐）
JPG
```

### 文件大小：
```
建议不超过 5MB
```

---

## 🎨 封面图设计方案

### 方案1：纯文字封面（最简单）

**背景颜色：**
```
深蓝色：#1a1a2e 或 #16213e
深紫色：#0f0f1a
深灰色：#1f1f2e
```

**文字布局：**

```
┌─────────────────────────────┐
│                           │
│     ⚔️                    │
│                           │
│   ENDLESS DUNGEON          │
│                           │
│  Roguelike Adventure       │
│                           │
│ ─────────────────────────  │
│                           │
│  • Endless Floors         │
│  • Equipment Grinding     │
│  • Auto Battle           │
│  • Free to Play          │
│                           │
│   RPG • Roguelike        │
│   Text-based            │
│                           │
└─────────────────────────────┘
```

**文字样式：**

标题 "ENDLESS DUNGEON":
- 字体：粗体，大号
- 颜色：白色 (#ffffff) 或 金色 (#ffd700)
- 字体大小：主标题占画面 40%

副标题 "Roguelike Adventure":
- 字体：中号
- 颜色：浅蓝色 (#64b5f6) 或 浅灰色 (#c0c0c0)
- 字体大小：副标题占画面 15%

特色列表:
- 字体：小号
- 颜色：浅灰色 (#a0a0a0)
- 符号：圆点或其他符号分隔

底部标签:
- 字体：极小号
- 颜色：深灰色 (#606060)

## 🛠️ 制作工具推荐

### 在线工具（免费，无需下载）：

#### 1. Canva（最推荐）
- 网址：https://www.canva.com/
- 优点：有现成模板，拖拽操作
- 操作步骤：
  1. 注册免费账号
  2. 搜索 "YouTube thumbnail" 或 "Social Media Graphic"
  3. 选择模板
  4. 修改文字和颜色
  5. 导出为 PNG

#### 2. Figma
- 网址：https://www.figma.com/
- 优点：专业设计工具
- 操作步骤：
  1. 注册免费账号
  2. 创建新画布（640x360）
  3. 添加文字和图形
  4. 导出为 PNG

#### 3. Photopea（类似Photoshop）
- 网址：https://www.photopea.com/
- 优点：功能强大
- 操作步骤：
  1. 打开网站
  2. 新建文档（640x360）
  3. 添加文字和图层
  4. 导出为 PNG

### 本地软件：

#### 1. PowerPoint / Keynote（最简单）
- 新建幻灯片
- 设置尺寸为 640x360
- 添加文字和图形
- 导出为图片

#### 2. Paint（Windows自带）
- 新建画布（640x360）
- 手动绘制或添加文字
- 保存为 PNG

## 📝 分步制作教程（使用 Canva）

### 第1步：访问 Canva
```
打开：https://www.canva.com/
```

### 第2步：注册账号
```
1. 点击 "Sign up"
2. 使用邮箱或 Google 账号注册
3. 选择免费计划
```

### 第3步：创建设计
```
1. 点击 "Create a design"
2. 选择 "YouTube Thumbnail" (1280x720)
3. 或选择 "Custom Dimensions" 输入 640x360
```

### 第4步：添加背景
```
1. 点击左侧 "Elements" → "Shapes"
2. 选择矩形
3. 设置颜色为深蓝色 (#1a1a2e)
4. 拖满整个画布
```

### 第5步：添加主标题
```
1. 点击左侧 "Text"
2. 选择 "Add heading"
3. 输入："ENDLESS DUNGEON"
4. 设置字体为粗体
5. 颜色设为白色或金色
6. 调整大小和位置（居中偏上）
```

### 第6步：添加副标题
```
1. 再次点击 "Text" → "Add subheading"
2. 输入："Roguelike Adventure"
3. 颜色设为浅蓝色 (#64b5f6)
4. 调整大小和位置（主标题下方）
```

### 第7步：添加特色列表
```
1. 点击 "Text" → "Add body text"
2. 输入特色列表
   • Endless Floors
   • Equipment Grinding
   • Auto Battle
   • Free to Play
3. 调整字体大小和位置（右侧）
```

### 第8步：添加图标（可选）
```
1. 点击 "Elements" → "Graphics"
2. 搜索："sword" "dungeon" "gem"
3. 选择合适的图标
4. 调整大小和位置
```

### 第9步：调整和美化
```
1. 检查所有文字是否清晰
2. 调整颜色对比度
3. 确保信息层次分明
4. 添加装饰元素（线条、图形等）
```

### 第10步：导出
```
1. 点击右上角 "Download"
2. 选择格式：PNG
3. 质量：标准
4. 点击 "Download" 下载
```

## 📐 快速制作（5分钟）

如果你不想花太多时间，这里是超快速方法：

### 使用 PowerPoint：

```
1. 打开 PowerPoint
2. 新建幻灯片
3. 右键 → 幻灯片大小 → 自定义
4. 宽度：640，高度：360（单位：像素）
5. 设置背景为深蓝色
6. 添加文本框
7. 输入："ENDLESS DUNGEON"
8. 设置字体：粗体，白色
9. 调整大小居中
10. 添加副标题："Roguelike Adventure"
11. 设置颜色：金色
12. 保存为图片 → PNG
```

### 使用 Paint（Windows）：

```
1. 打开画图
2. 文件 → 属性 → 宽度640，高度360
3. 用油漆桶填充背景为深蓝色
4. 选择文字工具
5. 输入："ENDLESS DUNGEON"
6. 设置为白色，粗体
7. 保存为 PNG
```
