# 游戏功能更新完成

## ✅ 已实现的功能

### 1. 🏆 排行榜系统
- 死亡时自动上传成绩到排行榜
- 排行榜存储在本地localStorage
- 显示前50名玩家记录
- 按层数优先排序，相同层数按金币排序

**位置：**
- 新增"🏆 排行榜"按钮
- 点击可查看排行榜界面

### 2. 💀 死亡逻辑优化
- **接受死亡：** 回到第1层，所有数据初始化（装备、遗物、法术清空）
- **看广告复活：** 原地复活，获得5分钟属性翻倍Buff（攻击×2、防御×2）

### 3. ⚡ 自动法术触发系统
法术改为自动触发，无需手动点击：

- **攻击法术：** 每次攻击时自动释放最高级的攻击法术
  - 优先选择最高等级的攻击法术
  - 自动消耗法力
  - 法力不足时不释放

- **治疗法术：** 血量低于50%时自动释放最高级治疗法术
  - 优先选择最高等级的治疗法术
  - 自动消耗法力
  - 法力不足时不释放

### 4. 🛒 批量购买系统
商店按钮根据金币自动显示批量购买选项：

- **金币 < 5倍基础价格：** 单次购买
- **金币 ≥ 5倍：** x5批量购买
- **金币 ≥ 10倍：** x10批量购买
- **金币 ≥ 20倍：** x20批量购买
- **金币 ≥ 50倍：** x50批量购买
- **金币 ≥ 100倍：** x100批量购买

**示例：**
- 初始金币 < 100：购买按钮显示"🏪 随机装备 (20💰)"
- 金币 ≥ 500：购买按钮显示"🏪 随机装备 (100💰 x5)"
- 金币 ≥ 2000：购买按钮显示"🏪 随机装备 (2000💰 x100)"

治疗按钮同样支持批量购买。

## 📊 游戏体验优化

1. **更流畅的战斗：** 法术自动释放，无需手动操作
2. **更便捷的购买：** 一键批量购买，省时省力
3. **更有挑战的死亡：** 死后必须从第一层重新开始
4. **更有价值的复活：** 看广告获得强力Buff继续挑战
