# itch.io HTML5 游戏上传指南（更新版）

## 🎯 重要说明

itch.io 上传 HTML5 游戏有 **两种方式**：

### 方式 1：上传 HTML 文件包（推荐）
- 打包整个游戏文件夹
- 上传到 itch.io
- 玩家在 itch.io 上直接玩

### 方式 2：外部链接（你现在的情况）
- 只填游戏信息
- 提供外部链接
- 玩家点击后跳转到你的网站

---

## 📋 如果用方式 2（外部链接）：

### 你现在看到的字段：

1. **Game title** - 游戏标题
   ```
   Endless Dungeon - 无尽地下城
   ```

2. **Short description** - 简短描述
   ```
   A text-based Roguelike adventure with endless floors and equipment grinding. Free to play!
   ```

3. **Kind of project** - 项目类型
   ```
   选择：HTML
   ```

4. **Embed width / Embed height** - 嵌入尺寸（可选）
   - 填：Width: 800, Height: 600

---

## 🔑 外部链接在哪里？

### 找到这个字段：
- 滚动到页面下方
- 找到 **"Embed URL"** 或 **"External URL"** 字段
- 填入你的游戏地址：
   ```
   https://devilmaycry19960807-coder.github.io/
   ```

**或者找到 "This game can be played on another website" 部分填写**

---

## 🚀 如果找不到外部链接字段，换用方式 1：

### 打包游戏上传：

1. **准备游戏文件包**
   - 找到你的游戏文件夹：`roguelike/`
   - 把整个文件夹压缩成 `.zip`
   - 文件名：`endless-dungeon.zip`

2. **上传文件**
   - 在 itch.io 页面找到 **"Upload files"** 或 **"Upload builds"**
   - 选择你的 `endless-dungeon.zip`
   - 等待上传完成

3. **发布**
   - 填写标题和描述
   - 点击 **"Publish"** 或 **"Save"**

---

| 方式 | 耗时 | 优点 |
|------|------|------|
